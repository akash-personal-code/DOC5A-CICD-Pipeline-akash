# Dashboard Wireframes

## Engineering Dashboard

- Current deployment colour/version
- Canary traffic percentage
- HTTP 5xx rate
- p95/p99 latency
- DB connection pool usage
- RabbitMQ queue depth
- Rollback status

## Management Dashboard

- Deployment frequency
- Lead time for changes
- Change failure rate
- MTTR
- Gate pass/fail trends
- Release calendar and blackout windows

## Regulatory Dashboard

- Change ID to deployment evidence mapping
- SAST/DAST/Trivy/SBOM evidence status
- Approval records
- Exceptions and expiry dates
- RBI and PCI control mapping
