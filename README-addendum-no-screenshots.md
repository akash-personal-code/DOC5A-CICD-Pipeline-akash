## Evidence Note

Screenshots were not available for every tool at the time of packaging. The repository therefore uses file-based evidence: source configuration, generated outputs, validation logs, policy files, exception records, dashboard JSON, and TRC PDF evidence.

The `evidence/screenshots/` folder is retained with a README and screenshot-unavailability note. Fresh screenshots can be added later from the same command outputs listed in `evidence/test-results/commands-to-generate-evidence.md`.
