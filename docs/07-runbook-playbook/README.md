# Runbook and Incident Playbook

This deliverable is stored in the root `runbooks/` directory:

- `runbooks/deployment-runbook.md`
- `runbooks/incident-playbook.md`

The deployment runbook defines the production deployment procedure, pre-deployment checklist, blue-green/canary execution, rollback decisions, post-deployment verification, and evidence requirements.

The incident playbook defines incident severity, escalation paths, response workflow, communication templates, postmortem format, and the Friday 5 PM incident simulation response.
